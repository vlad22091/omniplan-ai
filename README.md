# OmniPlan AI 🚀

OmniPlan AI — це інтелектуальний веб-додаток для генерації та управління планами будь-якої складності. Від розкладу тренувань до архітектурних проектів — сервер використовує нейромережу для створення чітких, структурованих кроків, а також дозволяє відстежувати прогрес їх виконання.

## 🌟 Основний функціонал

- **AI Генерація Планів:** Інтеграція з Groq API (модель Llama-3) для швидкого створення детальних планів на основі користувацьких промптів.
- **Збереження даних:** Надійна робота з базою даних завдяки безсерверному (serverless) PostgreSQL від Neon.
- **Трекінг статусів:** Можливість миттєво змінювати статус проекту (Планування 📝, Реалізація ⚙️, Завершено ✅) без перезавантаження сторінки завдяки AJAX/Fetch.
- **SSR (Server-Side Rendering):** Візуальний інтерфейс побудований за допомогою шаблонізатора Handlebars (`hbs`).

## 🛠 Технологічний стек

- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL (через `pg` пакет), керована в Neon DB
- **AI Integration:** Groq API
- **Frontend:** HTML/CSS + Handlebars (`hbs`) + Vanilla JS (Fetch API)
- **Інструменти:** `dotenv` для безпеки, `express-async-errors` для обробки помилок, `nodemon` для зручної розробки.

## 🗄 Структура бази даних

Таблиця `plans` створюється автоматично при першому підключенні до бази. Структура:
- `id` (SERIAL PRIMARY KEY)
- `title` (VARCHAR) - Назва плану
- `category` (VARCHAR) - Категорія (Спорт, Будівництво, Навчання тощо)
- `user_prompt` (TEXT) - Початковий запит користувача
- `generated_content` (TEXT) - Відповідь від ШІ
- `is_favorite` (BOOLEAN) - Позначка улюбленого плану
- `status` (VARCHAR) - Поточний стан (за замовчуванням 'Планування')
- `created_at` (TIMESTAMP)
